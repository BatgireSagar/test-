# Ops UI

A simple React-based user interface for KYC Operations Analysts and Compliance Officers to review onboarding cases, validation reports, and take manual actions.

## Features
- List onboarding cases and their validation status
- View validation reports and audit trails
- Claim/review tasks, approve/reject, and add comments

## Requirements
- Node.js 18+
- npm or yarn

## Running Locally
```sh
cd ops-ui
npm install
npm start
```

The app will be available at [http://localhost:3000](http://localhost:3000)
