# Camunda Workflow

This module contains BPMN and DMN files, configuration, and documentation for the KYC onboarding process orchestration using Camunda 8.

## Contents
- `bpmn/` — BPMN process definitions
- `dmn/` — DMN decision tables for business rules
- `docs/` — Workflow documentation and diagrams

## Purpose
- Orchestrate the end-to-end KYC onboarding process
- Integrate with external services (OCR, LLM, Validation, Manual Review)
- Support human-in-the-loop and compliance review

## How to Use
- Deploy BPMN and DMN files to Camunda 8
- Update process definitions as business requirements evolve
