# Sample Data

This directory contains example Policy & Procedure (PP) documents and client onboarding profiles for testing the KYC onboarding automation system.

## Contents
- `pp-doc-sample.txt` — Example PP document (text)
- `client-profile-sample.json` — Example client onboarding package (JSON)

## Usage
- Use these files to test the OCR, LLM/Rule Engine, and Validation Engine services.
- Update or add new samples as needed for additional scenarios.
